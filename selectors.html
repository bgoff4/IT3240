Slectors in style sheet

Selectors are used to indicate the HTML element to which we want to apply the style. A selector can be an element name, attribute, class, id etc.

Id selector

When we use id selecor, the style will be applied to the element which have the specified id.

Id attribute

Id attribute gives a unique id for an html element.

Class selector

Class selector uses class attributes of the elements to apply the style. All the elements having that class name will be formatted according to the defined rule.

Class attribute

Class attribute will group the elements with the same class name.
