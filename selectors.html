Selectors in Style Sheets

Selectors are used to indicate the HTML element to which we want to apply the style. A selector can be an element name, attribute, class, id, etc.

Id Selector

When we use an id selector, the style will be applied to the element which has the specified id.

Id Attribute

The id attribute gives a unique id for an HTML element.

Class Selector

Class selectors use class attributes of the elements to apply the style. All the elements having that class name will be formatted according to the defined rule.

Class Attribute

The class attribute will group the elements with the same class name.
